## 🌍 The Problem
African institutions (banks, fintechs, and remittance providers) face **3–7 day settlement delays** and **2–4% friction costs** when moving capital across borders. Legacy SWIFT rails are slow, and standard crypto rails lack the institutional guardrails, compliance, and multi-sig governance required for regulated operations.

## 🏛️ The Solution (CorridorOS)
CorridorOS is a Solana-based institutional treasury infrastructure protocol that enables compliant, real-time cross-border stablecoin transfers (USDC/USDG) with embedded KYC/KYT/AML and Travel Rule enforcement.

### Key Pillars:
- **Permissioned Liquidity**: Institutions deposit into escrow vaults keyed by origin/destination corridors (e.g., NG-USDC → KE-USDC).
- **Embedded Compliance**: Mandatory KYC verification and real-time transaction monitoring (KYT).
- **Blockchain Finality**: Atomic settlements on Solana with sub-second finality and negligible fees.
- **Institutional Governance**: Multi-sig approval workflows and IP-whitelisted access control.

---

## ✨ Features

- **🚀 Atomic Settlement Rail**: High-throughput settlement engine with real-time status tracking (Idle → Validating → Signing → Settling → Success).
- **🛡️ Risk & Audit Hub**: Comprehensive compliance dashboard featuring active KYT risk vectors, AI-driven SAR insights, and IVMS101 Travel Rule handshakes.
- **💎 Luxury UX/UI**: High-fidelity, glassy institutional interface built with **React 19**, **Tailwind v4**, and **Framer Motion**.
- **📊 Global Liquidity Explorer**: Visual performance metrics for regional corridors (Nigeria, Kenya, EU, etc.).
- **🔐 Multi-Sig Governance**: Threshold-based signature requirements for principal capital movements.

## 🏃 Getting Started

### Prerequisites
- Node.js (v18+)
- npm / pnpm / yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/eddiemessiah/corridor.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
